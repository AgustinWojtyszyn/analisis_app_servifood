Resumen de cambios
- Se redisenio completamente la landing publica de ServiFood en el componente existente.
- Se reemplazo la composicion anterior por una entrada institucional compacta con identidad corporativa mas marcada.
- Se agrego un header institucional con logo, texto Plataforma interna y botones publicos.
- Se creo un hero en dos columnas para desktop con composicion visual propia en HTML, CSS e iconos existentes.
- Se reorganizo el contenido en tres pilares del ecosistema ServiFood, sin crear un catalogo de modulos.
- Se agrego una seccion integrada de experiencia segun perfil sin selector de rol ni accesos separados.
- Se mantuvo la navegacion publica existente mediante callbacks onLogin y onRegister.
